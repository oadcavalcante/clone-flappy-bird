## Features

- ✨ Gameplay de Flappy Bird no browser
- 🚀 Controle por clique ou tecla para voar
- ⚡ Colisão com canos e chão
- 🎯 Pontuação progressiva
- 🔧 Sprites e animações em CSS/JS
- 📦 Sem dependências ou build step
